# Auprolis - The Sheriff Asset Marketplace of Botswana

A static website for the Auprolis asset marketplace platform.

## Landing Page Features

- **Responsive Design**: Mobile-first approach with clean, modern styling
- **Navigation**: Fixed navigation bar with smooth scrolling
- **Hero Section**: Eye-catching hero with call-to-action buttons
- **Features Section**: Highlights key platform benefits
- **Pricing Plans**: Simplified Free/Paid tier structure
- **Featured Assets**: Showcase of available assets
- **Process Explanation**: Step-by-step guides for buyers and sellers
- **Interactive Elements**: Tab switching, mobile menu, scroll animations
